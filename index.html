Linux CLI + GUI for Android

Experience the power of Linux on your Android device with LinuxDroid. Run a variety of Linux distributions with both command-line and graphical interfaces, no root required.

Get Started GitHub

Quick Installation

Get started with our one-click installation script:

curl -sSL https://raw.githubusercontent.com/AryanVBW/LinuxDroid/refs/heads/main/1click.install.sh | bash

For more detailed installation instructions, visit our installation guide.

Key Features

Multiple Distributions

Choose from a wide range of Linux distributions including Ubuntu, Kali Linux, Parrot OS, Arch Linux, and more.

GUI Support

Enjoy full graphical desktop environments like LXDE, XFCE, MATE, and even KDE Plasma on your Android device.

No Root Required

Run Linux distributions without rooting your Android device, preserving your warranty and device security.

Full Tool Compatibility

Access popular Linux tools like Nmap, Wireshark, Metasploit, and more for development, penetration testing, and everyday use.

Window Managers

Choose from lightweight window managers like Awesome, Openbox, and i3 for a customized desktop experience.

Easy Installation

Get up and running quickly with our one-click installation scripts and user-friendly setup process.
